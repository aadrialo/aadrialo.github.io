¿Qué es Odoo?

Odoo o también conocido como Odoo ERP es un software de gestión empresarial de código abierto. Permite una total integración y personalización, ofreciendo una solución para supervisar diversos aspectos de las operaciones de e-commerce.

Entre sus funcionalidades destacadas se encuentran la gestión de ventas y marketing, relaciones con clientes, supervisión de proyectos, fabricación, control de inventario, contabilidad, administración de recursos humanos, gestión de servicio al cliente y otras aplicaciones.

Aquí se muestran los pasos para instalar Odoo:

1.

Entramos en la pagina oficial de Odoo

2.

Bajamos hasta el fondo de la pagina y en "Codigo Abierto" le daremos a "Descargar"

3.

Seleccionamos nuestro sistema operativo y descargamos el "Community"

4.

Una vez descargamos iniciamos el .exe que descargamos

5.

Seleccionamos instalar Odoo y PostgreSQL

6.

Seleccionamos la configuracion para PostgreSQL

7.

Instalamos Odoo

8.

Cuando acabe la descarga podremos empezar a utilizar Odoo

9.

Arriba derecha si le damos, podremos descargar la app por si no quieres trabajar desde la web
